# 用户自定义知识库配置

> 本文件用于配置用户自定义的知识库目录路径
> 技能在分析问题时将自动扫描这些目录，匹配相关案例和参考资料

---

## 知识库目录配置

### 格式说明
- 每行一个目录路径
- 支持绝对路径和相对路径（相对于本配置文件所在目录）
- 以 `#` 开头为注释
- 空行会被忽略

### 示例格式
```
# 案例库
D:\AudioCases

# 常见问题汇总
D:\AudioFAQ

# 供应商文档
D:\VendorDocs\Qualcomm
```
